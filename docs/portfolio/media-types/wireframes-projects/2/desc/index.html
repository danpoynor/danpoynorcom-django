Portfolio / Media Types / Wireframes

Wireframes are useful for communicating what information will appear on a page and how it will be grouped and organized (the “visual hierarchy”). They’re similar to an architectural floor plan which is used as a planning tool to document the layout and relationships of physical features and objects before a building is constructed. Wireframes can represent different levels of detail such as a 10,000 foot overview showing general organization or a high-fidelity detailed mockup showing the most detail.
